# 🎅🧩🎄 Advent of Code 2024 🎄🧩 🎅

![Build Status](https://github.com/ariejan/advent-of-code-2024/actions/workflows/tests.yml/badge.svg)

Trying to be prepared for this year's edition...

## Setup

```sh
git clone git@github.com:ariejan/advent-of-code-2024
bundle install
```

## Generating new day

```
bin/generate <day>
```

## Running tests

```sh
rspec
```

## Solving problems

```
bin/aoc <day>
bin/aoc all
```
